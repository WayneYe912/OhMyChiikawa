MyBuddy - 安装说明 / Installation Guide
======================================

中文
----

1) 安装 MyBuddy
   把左边的 MyBuddy 图标拖到右边的「应用程序 / Applications」文件夹。

2) 首次打开
   macOS 可能会提示无法确认开发者。
   解决办法(只需做一次):
     在「应用程序」里【右键点击 MyBuddy → 选择"打开" → 再点"打开"】。
   之后即可正常双击启动。

   如果仍打不开,可在「终端」运行一次:
     xattr -dr com.apple.quarantine /Applications/MyBuddy.app

3) (可选)安装命令行工具
   如果你希望在终端使用 mybuddy 命令启动应用,双击本安装包里的
   「install-cli.command」。

   安装后可使用:
     mybuddy                 启动 MyBuddy
     mybuddy --scale=small   小尺寸
     mybuddy --scale=medium  中尺寸
     mybuddy --scale=large   大尺寸

--------------------------------------------------------------------
日常用法
  · 拖动乌萨奇      移动位置
  · 单击身体        跳一下并随机说话
  · 单击耳朵/脸     耳朵抖动 / 头部点头
  · 双击            转手动作
  · 右键            打开菜单(跟随鼠标、四处走动、尺寸、退出...)
退出:右键乌萨奇,选择退出菜单。


English
-------

1) Install MyBuddy
   Drag the MyBuddy icon into the Applications folder.

2) First launch
   macOS may ask for extra confirmation.
   To open it the first time:
     In Applications, right-click MyBuddy, choose "Open", then confirm "Open".
   This is usually needed only once.

   If it still cannot open, run this in Terminal:
     xattr -dr com.apple.quarantine /Applications/MyBuddy.app

3) Optional command-line launcher
   If you want to launch MyBuddy from Terminal, double-click
   "install-cli.command" in this installer.

   After installation, you can use:
     mybuddy                 Launch MyBuddy
     mybuddy --scale=small   Small size
     mybuddy --scale=medium  Medium size
     mybuddy --scale=large   Large size

--------------------------------------------------------------------
Basic usage
  - Drag usagi       Move it around
  - Click body       Hop and say a random line
  - Click ear/face   Wiggle ear / nod head
  - Double-click     Play the hand-rolling action
  - Right-click      Open the menu (follow cursor, wander, size, quit...)
Quit: right-click usagi and choose the quit menu item.
